# file_util

| 方法            | 描述                                               |
| --------------- | -------------------------------------------------- |
| init            | 初始化 FileUtil                                    |
| isDirFileExsit  | 判断文件或者目录是否存在                           |
| makeDir         | 创建文件夹；如果不存在则创建，如果存在则不进行操作 |
| backup_file     | 备份文件                                           |
| writeFile       | 写入文件                                           |
| readFile        | 读取文件                                           |
| getTempDir      | 获取临时文件夹路径                                 |
| dirIsEmpty      | 判断文件夹是否为空                                 |
| deleteDirOrFile | 删除文件或文件夹                                   |
| deleteDir       | 删除文件夹                                         |
| deleteFile      | 删除文件                                           |
